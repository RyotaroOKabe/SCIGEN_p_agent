# Evaluation Scripts

This directory contains scripts for evaluating generated crystal structures.

## Scripts

- **`eval_elem_ptable.py`**: Periodic table element analysis and evaluation

## Usage

```bash
python scripts/evaluation/eval_elem_ptable.py
```

## See Also

Core evaluation functionality is in `../script/eval_*.py`:
- `eval_screen.py`: Stability screening
- `eval_utils.py`: Evaluation utilities
- `eval_funcs.py`: Evaluation functions
