From the SCIGEN_p_agent directory:

## Available Papers

1. **technical_terms.md** - Definitions and notation
2. **key_concepts.md** - Main ideas and methods
3. **code_correspondence.md** - Paper-to-code mapping
4. **implementation_plan.md** - Implementation tasks
